═══════════════════════════════════════════════
2. IAAI (Insurance Auto Auctions) — Encheres salvage
═══════════════════════════════════════════════

IAAI a un portail B2B avec API pour acceder aux vehicules salvage/
accidentes. Ils ont aussi le DDR (Dynamic Data Repository) — un
service par abonnement qui donne acces electronique a TOUTES les
donnees de vehicules non-vendus, 24/7.

Ce qu'on a besoin:
  - Historique des vehicules accidentes vendus aux encheres
  - Photos et donnees de dommages
  - Prix de vente salvage (pour comparer avec la valeur marche)

Options d'acces:
  a) API officielle IAAI — via B2B Connect Portal
     → https://b2b-portal.iaai.com/
     → Inscription requise, contacter IAAI pour les prix

  b) API tierce (Copart + IAAI combine) — auction-api.app
     → https://auction-api.app/en/
     → Light: $200 USD/mois (30K requetes)
     → Medium: $300 USD/mois (100K requetes)
     → PRO: $500 USD/mois (500K requetes)
     → Inclut Copart + IAAI dans un seul abonnement

  c) Autre option tierce: carstat.dev
     → https://carstat.dev/
     → 20M+ records historiques d'encheres

Question: Est-ce que tu as deja un acces dealer a IAAI?
Si oui, est-ce que tu peux te connecter au B2B Connect Portal?

3. CANADIAN BLACK BOOK — Valeur vehicules usages
═══════════════════════════════════════════════

C'est LE standard de l'industrie au Canada pour evaluer les vehicules.
Ils ont 6 APIs:

  1. Used Car Web API — valeurs wholesale, trade-in, retail, residuel
  2. Retail Market Insights — 4,700+ concessionnaires, 700K+ annonces
  3. Custom Trade Value — evaluation personnalisee par condition
  4. New Car Web API — MSRP, prix facture, residuels
  5. Recreational Vehicle API — VR
  6. Cars of Particular Interest — luxe/collection depuis 1946

Pricing: a partir de $65/mois pour le plan de base
         APIs = cout supplementaire (sur devis)

IMPORTANT — est-ce que tu as deja un acces Canadian Black Book
via ton reseau dealer? Beaucoup de concessionnaires l'ont inclus
dans leur abonnement DealerTrack ou vAuto.

Portail developpeur: https://developer.canadianblackbook.com/
Info API: https://www.canadianblackbook.com/api/
Support: support@canadianblackbook.com
Contact: https://www.canadianblackbook.com/contact/
